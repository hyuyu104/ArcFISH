Differential 3D genome structures#

This notebook contains the code to reproduce Figure 5 and S18-S20.

Required data are:

  1. Takei et al, 2021, Nature: 25 Kb dataset from two biological replicates

    data.json: takei_nature_2021, 25Kb, rep1 and rep2

    4DN data portal ID: 4DNFIHF3JCBY (rep1), 4DNFIQXONUUH (rep2)

  2. Takei et al, 2021, Science: 25 Kb dataset from three biological replicates

    data.json: takei_science_2021

    cell type information is available with the key cell_type

  3. ChIP-seq data of mouse embryonic stem cells

    data.json: chipseq_mesc

  4. Single-cell RNA-seq data of mouse cerebral cortex (PMID: 30635555)

    Count matrix and cluster annotation are downloaded from jeremymsimon/MouseCortex

    Detailed processing is available in 40rnaseq.ipynb

  5. Single-cell RNA-seq data of mouse embryonic stem cell (PMID: 26804902)

    RPKM matrix is downloaded from paper supplementary

    Detailed processing is available in 40rnaseq.ipynb

def fetch_adata(chr_id, cell_type):
    adata = loader.create_adata(chr_id)
    af.pp.add_cell_type(adata, celldfs, "Cell_ID", "cluster label")
    celltype_dict = {1:'Pvalb', 2:'Vip', 3:'Ndnf', 4:'Sst', 5:'Astro',
                    6:'Micro', 7:'Endo', 8:'Oligo', 9:'Ex'}
    adata.obs["cell_type"] = adata.obs["cluster label"].map(celltype_dict)
    return adata[adata.obs["cell_type"] == cell_type].copy()
